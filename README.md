# DinamicalSystems

Im this project is tested some hybrid systems, using IBEX C++ and DynIbex. 
In order to find patterns which control the system behaviour.

#### Compiling and Running


``` cmd
cd [example_name]/build/
mkdir build
cd build
cmake ..
make && ./out
```
